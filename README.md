# Flutter肺炎疫情应用

## 简介
肺炎疫情应用是一款肺炎工具类查询应用，提供肺炎疫情最新的发展信息，以及提供一些防护措施。此应用主要由首页、看一看、新闻和防护模块构成，提供最新的疫情实时数据。
